**SEO técnico para Next.js** é o conjunto de implementações no código — metadados,
dados estruturados, sitemaps, renderização e performance — que permite ao Google
rastrear, indexar e rankear um site feito com App Router. Diferente do SEO de
conteúdo, ele é resolvido no repositório: `Metadata API`, `sitemap.ts`,
`robots.ts`, JSON-LD e Core Web Vitals. Este guia mostra como implementar cada
peça, com código real e resultados medidos neste próprio site.
